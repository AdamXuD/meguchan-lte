# meguchan

> 惠酱的长期维护版本（目前正处于开发阶段）

## 如何部署

1. 安装 docker 与 docker-compose。
2. `docker-compose up`（开摆

## 功能

### emoji 表情包缝合

发送两个连续的 emoji 表情即可。

### 按 Pixiv ID 搜索 p 站图片

指令: 惠酱[p|P]站(\s)?(?P `<pid>`\d+)

示例: 惠酱 p 站 12345678

### 以图搜图

指令：惠酱搜图

然后按照引导继续即可。
